# Goldex Icons Library

Goldex Icons Library. Icons for components.

# Install

<pre>
npm i goldex-icon-library
</pre>

# Update 1.0.0

<ul>
 <li>Add icons: Lock, Unlock, Key, Spinner, Support</li>
</ul>

# Basic Usage

<pre>
import { LogoIcon } from 'goldex-icon-library'

export function Component() {
     
   return (
       
         < LogoIcon / >
   )  
}
</pre>
